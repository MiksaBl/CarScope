# Car Deal Analyzer
Analyzes car listings and compares them with market values to identify potentially good deals.

Features
- Monitors car listings
- Evaluates market value
- Sends Telegram notifications
- Desktop GUI built with Tkinter

Technologies
- Python
- Playwright
- SQLite
- Tkinter
- Telegram Bot API

Installation
pip install -r requirements.txt
playwright install
python car_deal_analyzer.py
